## Run These Commands (One at a time, in order)

### Command 1 - Set your email:
```bash
git config --global user.email "your-email@gmail.com"
```

### Command 2 - Set your name:
```bash
git config --global user.name "Your Name"
```

### Command 3 - Go to your project:
```bash
cd C:\Users\Dell\Downloads\Discon
```

### Command 4 - Add all files:
```bash
git add .
```

### Command 5 - Create commit:
```bash
git commit -m "Initial commit: PowerTrack app"
```

### Command 6 - Rename branch:
```bash
git branch -M main
```

### Command 7 - Push to GitHub:
```bash
git push -u origin main
```

---

## ✅ Success = You see this:
```
[new branch] main -> main
Branch 'main' set up to track remote branch 'main' from 'origin'.
```
